JavaScript might not be enabled





        If the JavaScript is not enabled in your internet browser, you will need to enable it in order to be able to proceed.
        

        To enable JavaScript in your internet browser, follow the instructions below: 
        
	Internet Explorer 5 or later
 
Click Tools menu, click Internet Options
Click Security tab
Click Custom Level button
Scroll down to Scripting 

                Select Enable radio button under Active scripting,
Allow paste 
				operations via script (Allow Programmatic clipboard access in IE7) 
and Scripting of Java applets. 

Click OK
Click "Return" at the end of this page.


Mozilla Firefox browser 
 
Click Tools menu, click Options
Click Content tab
Select Enable JavaScript check box
Click OK
Click "Return" at the end of this page



For more information about JavaScript features, please consult the Help of your Internet browser.
